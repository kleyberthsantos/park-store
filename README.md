## Uso del proyecto

Clonar el repositorio de GitHub:
git clone https://github.com/kleyberthsantos/park-store.git

cd dentro del directorio de la carpeta:
cd park-store

Instala las dependencias.
npm install

Para correr el proyecto:
npm run dev
